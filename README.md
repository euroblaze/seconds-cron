Odoo 18 module for sub-minute scheduled actions.

## Description

Standard Odoo cron jobs only support minute-level scheduling. This module enables second-level cron execution for time-critical operations.

## Features

- Schedule cron jobs in seconds intervals
- Optimized for quick execution tasks

## Requirements

- Odoo 18.0
- base_setup module

## Installation

1. Download and extract to your Odoo addons folder
2. Update apps list in Odoo
3. Install "Run Scheduler Per Second"

## Configuration

After installation, configure your scheduled actions to use second-level intervals via the module settings.
